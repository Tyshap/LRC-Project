# LEFT CENTER RIGHT CONSOLE GAME

This is a simple game that simulates the popular left center right game.

I've created the code for this from scratch,, it uses simple linked lists and objects to keep track of people and turn order.

It's one of my first projects before I knew much about memory management, though there should in theory be no memory leaks so long as the scale of the game isn't rediculous such as having 1000 players.

To run this program, you should find a .exe file inside of the x64/Debug folder
